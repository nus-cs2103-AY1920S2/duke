# User Guide

## Usage

### `Todo`

Adds a new task.

Example of usage: 

`todo go to the store`

Expected outcome:

A new task will be added.

### `Deadline`

Adds a new task with a deadline.

Example of usage: 

`deadline Go to the store /by 2020-02-02`

Expected outcome:

A new task with a deadline will be added.

### `List`

Lists all tasks.

Example of usage: 

`list`

Expected outcome:

Duke will print all tasks.

### `delete`

Delete a task

Example of usage: 

`delete 1`

Expected outcome:

Duke will delete task 1.
